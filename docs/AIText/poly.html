    Poly - 制作纹理的AI

    # Poly 简介

    Poly允许您在几秒内使用简单的文本和/或图像提示，使用32位PBR贴图快速生成自定义、HD和无缝平铺纹理。创建新材质，从现有图像转换，生成变体，或探索使用Poly AI制作的数千种令人惊叹的纹理。

    # 使用方法

    打开网站登录账号，输入想要的材质描述，等待AI制作完成后，在编辑界面进行调整后下载。
